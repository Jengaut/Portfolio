Web Development

Snake project (website)

JSsnake is a game I created using JavaScript and CSS with a minimalist design. It is based on the classic Snake game, where the player must control a moving snake to avoid obstacles and eat food to grow.

Wordle project (website)

JSWordle is a word game based on Mastermind that I created using JavaScript and CSS with a minimalist design. Players must guess a word using the fewest number of attempts possible.

Tic-Tac-Toe project

My WordPress plugin is a tic-tac-toe game that allows users to play against each other. Players can play against another human player with different time limits.

Circuit project

The website I developed showcases a city that I selected. It is structured into three points of interest, a page dedicated to presenting the city, as well as a contact form that does not have a designated email recipient.
